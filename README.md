# BEFORE RUN PROJECT

<a href="https://docs.docker.com/desktop/setup/install/windows-install">install</a> docker

## After install open and write in bash
```bash
$ docker-compose up -d
```
## Set settings
- copy env_example
- paste to root
- rename to .env

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev
```
